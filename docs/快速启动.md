# 快速启动

## 一键启动

```bash
cd ~/vscode/taoke
./start.sh
```

或者：

```bash
cd ~/vscode/taoke
python3 run.py
```

启动后访问：**http://localhost:8000/admin**

## 关闭服务

终端按 `Ctrl+C` 即可停止。

## 开机后恢复定时任务

数据库中已创建的任务会自动恢复运行，无需重新配置。

1. 启动服务 `./start.sh`
2. 打开 http://localhost:8000/admin/schedules 确认任务状态为「▶ 运行中」
3. 如果是「⏸ 已暂停」，点击状态标签即可恢复

## 首次使用流程

1. 配置 `.env` 文件（大淘客 API key 等）
2. 启动服务
3. 账号管理 → 扫码登录微博
4. 商品管理 → 立即抓取
5. 选品发布 或 创建定时任务自动发布

## 电脑休眠说明

服务运行在你的电脑上，休眠后定时任务不会执行。如果要 7×24 运行，可以部署到 Railway 云端。

插电使用时，在「系统设置 → 电池 → 防止自动休眠」开启即可。
