# 配置说明

## .env 文件

项目根目录下 `.env` 文件，参考 `.env.example`：

```bash
# 大淘客 API（必填）
DATETOKE_APP_KEY=你的appKey
DATETOKE_APP_SECRET=你的appSecret

# 微博开放平台（可选，网页模拟登录不需要）
WEIBO_APP_KEY=your_weibo_app_key
WEIBO_APP_SECRET=your_weibo_app_secret
WEIBO_REDIRECT_URI=http://127.0.0.1:8000/api/weibo/callback

# AI 文案生成（可选，不配置则使用模板生成）
ANTHROPIC_API_KEY=your_anthropic_api_key
OPENAI_API_KEY=your_openai_api_key

# 数据库
DATABASE_URL=sqlite+aiosqlite:///./data/taoke.db

# 定时任务默认间隔
DEFAULT_FETCH_INTERVAL_HOURS=6
DEFAULT_POST_INTERVAL_HOURS=2
```

## 大淘客 API Key 获取

1. 注册/登录 https://www.dataoke.com
2. 进入开放平台 → 应用管理
3. 创建应用获取 AppKey 和 AppSecret

## 微博登录

使用**扫码登录**方式，无需配置 API Key：
1. 账号管理页 → 点击「扫码登录微博」
2. Playwright 弹出浏览器窗口
3. 用微博 App 扫码完成登录
4. Cookie 自动保存到数据库

如果微信/微博开放平台 API 可用，也可以使用 OAuth2 授权方式（需配置 `WEIBO_APP_KEY` 等）。

## 商品评分公式

```
score = min(100, log₁₀(月销量) × 15 + 佣金率 × 0.8)
```

- 月销 1 万 + 佣金 10% ≈ 68 分
- 月销 10 万 + 佣金 20% ≈ 83 分
- 月销 100 万 + 佣金 30% ≈ 99 分

## 定时任务 Cron 表达式

格式：`分 时 日 月 周`

| 表达式 | 含义 |
|--------|------|
| `0 * * * *` | 每小时整点 |
| `0 */6 * * *` | 每 6 小时 |
| `0 9,20 * * *` | 早 9 点、晚 8 点 |
| `0 9,12,16,20 * * *` | 每天 4 次 |
| `30 8 * * 1-5` | 工作日 8:30 |

## 项目结构

```

├── app/
│   ├── main.py              # FastAPI 入口
│   ├── config.py            # 配置管理
│   ├── database.py          # 数据库
│   ├── models/              # ORM 模型
│   ├── schemas/             # Pydantic 校验
│   ├── routers/             # API 路由 + 管理后台
│   ├── services/
│   │   ├── fetcher/         # 商品抓取（大淘客/淘宝/京东/拼多多）
│   │   ├── content/         # 文案生成 + 图片处理
│   │   ├── publisher/       # 发布器（微博 API / 网页模拟）
│   │   └── scheduler.py     # 定时任务引擎
│   ├── templates/           # Jinja2 管理后台页面
│   └── static/              # CSS 等静态资源
├── scripts/                 # CLI 工具
├── docs/                    # 文档
├── data/                    # SQLite 数据库 + 商品图片
├── .env                     # 配置文件（不提交 Git）
├── .env.example             # 配置模板
├── requirements.txt         # Python 依赖
├── run.py                   # 开发启动脚本
└── start.sh                 # 快捷启动
```
