# 功能说明

## 系统概览

自动带货系统 — 从大淘客获取高价值商品，生成推广文案，通过微博网页版自动发布。

## 模块

### 1. 选品库

- 从大淘客 API 抓取高价值商品（销量/佣金综合评分）
- 自动转链生成 CPS 推广链接 + 淘口令
- 三种文案风格：优惠信息型 / 内容推荐型 / AI 生成
- 一键复制：文案、CPS 链接、淘口令、全部
- 收藏管理

### 2. 账号管理

- 扫码登录微博（Playwright 模拟网页登录，Cookie 持久化）
- 无需微博开放平台 API 权限
- Cookie 有效性自动检测

### 3. 手动发布

- 选品库中点击「📤 发布」
- 选择微博账号 → 预览文案 → 确认发布
- Playwright 模拟手机浏览器操作：打字、上传图片、点击发送

### 4. 定时任务

- 快速预设：每6小时 / 每天4次 / 早晚各1次
- 自定义 Cron 表达式
- 发前自动刷新商品库（避免下架品）
- 去重：已发布过的商品不会重复发
- 暂停 / 启用 / 立即执行

### 5. 收益看板

- 大淘客订单查询 API
- 预估收入、已结算收入、订单数
- 按时间筛选（今天/昨天/近7天/近30天）
- 注意：需连续使用转链 API 3 天后才有订单数据

## 发布流程

```
商品抓取 → 转链(CPS链接+淘口令) → 文案生成
                                           ↓
            Playwright 模拟发博 ← 定时任务/手动触发
```

## 行为模拟（防检测）

- 逐字输入，80-250ms 随机间隔
- 句间随机停顿 0.3-1s
- 偶尔 200-600ms 犹豫停顿
- 随机页面滚动
- 手机 UA 伪装
- 反 webdriver 检测脚本
